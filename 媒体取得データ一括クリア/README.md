# 媒体集計フォルダ 一括データ消去スクリプト

Google Drive の「媒体集計フォルダ」配下にある全スプレッドシートに対して、
指定したシートの指定範囲の**値のみ**を一括消去するGoogle Apps Script（GAS）です。
書式・罫線・背景色はそのまま残ります。

---

## 対象フォルダ構成

```
数値集計フォルダ
└── 【ぐる・食べ・ホット】媒体レポート管理　　← このフォルダ以下が対象
    └── 各企業フォルダ
        └── 各店舗のスプレッドシート
```

---

## 消去対象シートと範囲（デフォルト設定）

各媒体管理画面からコピペするデータが削除対象です。

| シート名 | 消去範囲 |
|---|---|
| ぐるなび集計 | B3:N16 / B21:N34 / B39:N52 / B56:N69 |
| 食べログ集計 | B2:O15 / B19:O32 / B38:O50 / B55:O67 |
| ホットペッパー集計① | A17:L500 / P17:Z500 |

対象シートが存在しないスプレッドシートは自動的にスキップされます。

## セットアップ手順

### 1. GASプロジェクトを作成する

1. [https://script.google.com](https://script.google.com) を開く
2. 「新しいプロジェクト」をクリック
3. 表示されているコード（`function myFunction...`）を全削除
4. `gurutabe_clear_all.gs` の中身を貼り付けて保存

> ※ このスクリプトは**スタンドアロン型**です。特定のスプレッドシートに紐づける必要はありません。

### 2. フォルダIDを設定する

スクリプト上部の `CONFIG` にある `HOT_FOLDER_ID` に、
「媒体集計フォルダ」の **Google Drive フォルダID** を入力します。

```javascript
const CONFIG = {
  HOT_FOLDER_ID: 'ここにフォルダIDを入力',  // ← 変更する
  ...
};
```

**フォルダIDの確認方法：**
Google Drive でフォルダを開いたときのURLの末尾の文字列です。

```
https://drive.google.com/drive/folders/【この部分がフォルダID】
```

### 3. 権限を承認する

初回実行時に「このアプリはGoogleアカウントへのアクセスを求めています」という画面が表示されます。
「許可」をクリックしてください（Driveへの読み書き権限が必要です）。

## 使い方

### 本番実行

1. GASエディタで関数名を `clearAllSheets` に選択
2. 「実行」ボタンをクリック
3. 画面下部の「実行ログ」で進捗を確認

スプレッドシートが大量にある場合（目安：100件以上）、GASの実行時間制限（6分）に達すると自動的に一時停止します。
ログに以下のメッセージが表示されたら、再度 `clearAllSheets` を実行してください。続きから再開されます。

```
⏱ 時間制限（5分）に達しました。
   処理済み: 250 / 600 件
   残り: 350 件
➡ 「clearAllSheets」をもう一度実行すると続きから処理します。
```

### テスト実行（1件だけ試したい場合）

スクリプト内の `testSingleSpreadsheet()` 関数を使います。

1. スクリプト内の以下の行にテスト対象のスプレッドシートIDを入力

```javascript
const TEST_SPREADSHEET_ID = 'テストしたいスプレッドシートのIDを入力';
```

2. 関数名を `testSingleSpreadsheet` に選択して実行
3. ログで各シートの処理結果を確認

## 設定の変更方法

### ● 対象フォルダを変更したい

スクリプト上部の `HOT_FOLDER_ID` を書き換えます。

```javascript
HOT_FOLDER_ID: '新しいフォルダID',
```

### ● 消去するシートや範囲を変更したい

`TARGETS` 配列を編集します。シート名（`name`）と範囲（`ranges`）を自由に変更できます。

```javascript
TARGETS: [
  {
    name: "シート名",               // 変更したいシート名
    ranges: ["A1:Z100", "B5:D10"]  // 消去したいセル範囲（複数指定可）
  },
  ...
]
```

> **このスクリプト1つを変更するだけで全スプレッドシートに反映されます。**

---

## 関数一覧

| 関数名 | 説明 |
|---|---|
| `clearAllSheets()` | メイン処理。全スプレッドシートを一括消去する |
| `resetProgress()` | 処理の進捗をリセットして最初からやり直す |
| `testSingleSpreadsheet()` | 指定した1件のスプレッドシートだけ動作確認する |

---

## 注意事項

- 消去した値は元に戻せません。実行前にテスト用フォルダで動作確認することを推奨します。
- スクリプトを実行するGoogleアカウントが、対象フォルダおよびスプレッドシートへの編集権限を持っている必要があります。
- 消去されるのは**値のみ**です。書式（色・罫線・フォントなど）は保持されます。
