# 媒体集計スプレッドの一括セル変換

複数のGoogleスプレッドシートに対して、**特定シートの特定セルの数式を一括で書き換える** Google Apps Script（GAS）です。

---

## ⚡ 次回応用するときは「設定エリア」の3項目を変えるだけ

スクリプト上部の設定エリアを書き換えて実行するだけで、どんなシート・セル・数式にも応用できます。

```javascript
// ★ 設定エリア（ここだけ変更すればOK）

// 対象フォルダのID（GoogleドライブURLの folders/ 以降の文字列）
var FOLDER_ID = "1ZekCvwebZbioTV4sMLN2KfX0QxQnOxO7";

// 対象シート名
var TARGET_SHEET = "食べログ集計";

// 書き換える数式（セル: 数式 の形式で複数指定可能）
var FORMULAS = {
  "F74": '=VALUE(REGEXREPLACE(TO_TEXT(G71),"[¥￥, 　]",""))',
  "I74": '=VALUE(REGEXREPLACE(TO_TEXT(F72),"[¥￥, 　]",""))'
};

// サブフォルダも再帰的にスキャンするか（true=する / false=しない）
var SCAN_SUBFOLDERS = true;
```

**フォルダIDの確認方法：** GoogleドライブのフォルダURLが `https://drive.google.com/drive/folders/XXXXXXX` の場合、`XXXXXXX` 部分がフォルダIDです。

|---|---|

## 使い方

### 1. Google Apps Script を開く

[https://script.google.com](https://script.google.com) にアクセスし、「新しいプロジェクト」を作成します。

### 2. スクリプトを貼り付ける

`コード.gs` の中身をすべて削除し、使用するスクリプトの内容を貼り付けて保存（`Ctrl + S`）します。

### 3. 設定エリアを編集する

スクリプト上部の「設定エリア」を自分の環境に合わせて書き換えます（上記参照）。

### 4. 実行する

関数名が `fixFormulas` になっていることを確認して「▶ 実行」ボタンをクリックします。

> **初回実行時** は「Googleアカウントへのアクセスを許可」画面が表示されます。「許可」を選択してください。

### 5. 結果を確認する

「表示」→「ログ」で処理結果を確認できます。

```
==============================
処理完了
  成功: 74 件
  スキップ（シートなし）: 2 件
  エラー: 0 件
------------------------------
[OK] ○○店舗スプレッドシート
[SKIP] シートなし: △△店舗スプレッドシート
==============================
```

| ログ表示 | 意味 |
|---|---|
| `[OK]` | 数式の書き換えに成功 |
| `[SKIP]` | 対象シートが存在しないためスキップ |
| `[ERROR]` | アクセス権がない等のエラー |

## 動作の仕組み（フォルダスキャン版）

```
指定フォルダ
├── スプレッドシートA  ← 自動検出
├── スプレッドシートB  ← 自動検出
└── サブフォルダ/
    └── スプレッドシートC  ← SCAN_SUBFOLDERS=true なら検出
```

1. `DriveApp.getFolderById()` で指定フォルダを開く
2. フォルダ内のGoogleスプレッドシートをすべて取得（サブフォルダも再帰スキャン可）
3. 各スプレッドシートで対象シートを探し、存在すれば指定セルに数式を書き込む
4. 対象シートがないファイルは自動的にスキップ

---

## 注意事項

- スクリプトを実行するGoogleアカウントが、対象スプレッドシートすべてに **編集権限** を持っている必要があります。
- 対象ファイルが多い場合、GASの実行時間制限（6分）に引っかかる可能性があります。その場合は `SCAN_SUBFOLDERS = false` にしてフォルダを分割して実行してください。
- 書き換え前の数式には戻せないため、**必要に応じて事前にバックアップ**を取ることを推奨します。
